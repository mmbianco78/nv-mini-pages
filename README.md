# nv-mini-pages

Public mirror of the mini2 system-health report.

Source: private repo [`mmbianco78/nv-mini`](https://github.com/mmbianco78/nv-mini) → `pursuit:system-health-check` runs on mini2 (marcos-mac-mini-3), projects to `health.system_metrics` in local Postgres, then `scripts/render_health_report.py` regenerates `index.html` here.

Live view: https://mmbianco78.github.io/nv-mini-pages/
